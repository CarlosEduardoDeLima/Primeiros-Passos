Você pode encontrar o comprimento de um Stringvalor escrevendo .lengthapós a variável de string ou literal de string. console.log("Alan Peter".length); O valor 10seria exibido no console. Por exemplo, se criamos uma variável const firstName = "Ada", podemos descobrir o comprimento da string Adausando a firstName.lengthpropriedade.
