Também podemos usar o +=operador para concatenar uma string no final de uma variável de string existente. Isso pode ser muito útil para quebrar uma longa string em várias linhas. Nota: Cuidado com os espaços. A concatenação não adiciona espaços entre as strings concatenadas, portanto, você mesmo precisará adicioná-las. Exemplo: let ourStr = "I come first. "; ourStr += "I come second."; ourStragora tem um valor da string I come first. I come second..
