Quando as variáveis de JavaScript são declaradas, elas têm um valor inicial de undefined. Se você fizer uma operação matemática em uma variável undefined, seu resultado será NaN, o que significa que "Não é um número". Se você concatenar uma string com uma variável undefined, você receberá uma string com o valor undefined. Inicialize as três variáveis a, b e c com 5, 10, e "I am a" respectivamente para que eles não sejam undefined.
