Em JavaScript, quando o +operador é usado com um Stringvalor, ele é chamado de operador de concatenação . Você pode construir uma nova string a partir de outras strings concatenando -as. Exemplo 'My name is Alan,' + ' I concatenate.' Nota: Cuidado com os espaços. A concatenação não adiciona espaços entre as strings concatenadas, portanto, você mesmo precisará adicioná-las. Exemplo: const ourStr = "I come first. " + "I come second."; A string I come first. I come second.seria exibida no console.
