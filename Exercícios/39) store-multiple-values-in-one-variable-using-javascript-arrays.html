Com variáveis ​​JavaScript array, podemos armazenar vários dados em um só lugar. Você inicia uma declaração de array com um colchete de abertura, termina com um colchete de fechamento e coloca uma vírgula entre cada entrada, assim: const sandwich = ["peanut butter", "jelly", "bread"]; Modifique a nova matriz myArraypara que ela contenha uma string e um número (nessa ordem)
