A notação de colchetes é uma maneira de obter um caractere em um índice específico dentro de uma string. A maioria das linguagens de programação modernas, como JavaScript, não começa a contar em 1 como os humanos. Eles começam em 0. Isso é conhecido como indexação baseada em zero . Por exemplo, o caractere no índice 0 na palavra Charlesé C. Então se const firstName = "Charles", você pode obter o valor da primeira letra da string usando firstName[0]. Exemplo: const firstName = "Charles"; const firstLetter = firstName[0]; firstLetterteria um valor da string C.
