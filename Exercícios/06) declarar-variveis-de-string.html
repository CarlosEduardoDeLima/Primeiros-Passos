Anteriormente, você usou o seguinte código para declarar uma variável: var myName; Mas você também pode declarar uma variável string assim: var myName = "your name"; "your name" é chamado de string literal. Uma string literal, ou string, é uma série de 0 ou mais caracteres entre aspas simples ou duplas. Crie duas novas variáveis de string: myFirstName e myLastName e atribua a elas os valores do seu primeiro e último nome, respectivamente.
